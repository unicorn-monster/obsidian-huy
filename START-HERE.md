# START HERE — cách xài vault này

## Vòng lặp hằng ngày (bạn ~2 phút)
1. **Bắt:** thấy gì hay -> quăng vào `inbox/`. 3 cách:
   - Gõ ý tưởng: Obsidian `Cmd+N` -> gõ (tự vào inbox).
   - Bài web: bấm Obsidian Web Clipper trên Chrome.
   - Ảnh ad: kéo file thả vào folder `inbox/`.
   Không phân loại lúc bắt. Nghi ngờ -> cứ quăng inbox.
2. **Sáng:** mở Claude Code trong folder này, gõ `/brief`. Đọc. Trả lời câu hỏi nó hỏi.
3. **Sau mỗi test:** gõ 1 dòng `learning` vào inbox (cái gì win/chết + số).

## Khi rảnh
- `/ingest` — Claude tiêu hoá inbox vào `sources/` + bồi `wiki/`. Đừng để inbox phình.

## Cuối tuần
- `/synthesis` — ngồi 15 phút đọc tổng hợp tuần.
- Thứ 2: mở `CLAUDE.md`, update mục "Dự án đang chạy" + "Đang đọc & nghĩ". 5 phút.

## Mở mảng mới sau (finance / fulfillment)
Chỉ cần bắt đầu gắn `domain: finance` cho note mới. Không đổi cấu trúc gì.
