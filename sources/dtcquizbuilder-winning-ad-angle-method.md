How to crack a 'life-changing winning ad' methodically:

> Find new angle (Reddit / Facebook)
> Create Avatar
> Find 3 sub-angles
> Test ads for each
> Iterate one with best performance
> Create quiz

Client was testing a new angle, once it scaled we just wrote the quiz
