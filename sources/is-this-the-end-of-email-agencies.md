Recently I posted a tweet and some people were slightly unhappy. Now the only concrete evidence we have is whether this email works or not, which will be found out.

My beliefs on email agencies: majority of them suck. They focus on designs, that end up bringing in less revenue and overall costing you more in deliverability. My focus with this brand was writing good copy — and it led to the best email performance we've ever had: 40% rev while scaling, as well as launching some digital products to grab more profit.

## The Main Focus
Write good copy. Use Claude to convert this into a template with some CTAs. Have a preset HTML template — only header + footer. This made it easy to spin out new sections. This is important, as we had a ton of data to take into Claude Code in terms of overall layout & email structure.

## Digesting Data
We've sent 100's of emails. Some performed incredibly well, others not so much. So we got Claude Code to go through all the emails with the best open rates & revenue — identify the subject, identify the preview, what was the angle of the email?

## Bulk Ideation
Once again, another simple answer — but anything complicated here would be dumb. However you usually ideate emails, do this at scale: winning scripts? Turn into an email. New ad tests? Turn into an email. Winning emails? Shorten, make longer, or change them up a bit.

Take the email data you have, plug it into **Opus 4.6**, download **Wispr Flow**, and speak out all the ideas in depth. Opus 4.6 will then give you a list of all the email ideas, days to send them, subject lines, etc.

## Claude Code for Sending
This is where having a prebuilt template is very important. You'll need to have trained Claude Code on all your previous emails/templates etc. Essentially all you're going to do is copy Opus 4.6's answer into Claude Code. Make sure it included: subject line & preview line, links for CTAs, sending dates, segments to send to. This is all the information for Claude Code to not only create the email but schedule it live.

I am waiting for the test data, but I would suggest everyone take a 2-week pro-rata on whoever you work with. Test this method. See the results after.
