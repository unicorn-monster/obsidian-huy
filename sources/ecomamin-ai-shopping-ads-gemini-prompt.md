Ready-to-use Gemini/Nano-Banana-Pro prompt template for generating 5 conversion-optimized static image assets for Google Shopping + PMax, built to Google Merchant Center compliance rules.

Workflow tip: give ChatGPT loose product/brand info (+ a reference image) and ask it to fill in the blanks of this template first, then feed that filled output into the image model.

## Section 1 — e-commerce core data / compliance inputs
Brand name, product focus (must match feed Product Title), core USP/differentiator, price + anchor (current + crossed-out former MSRP), current offer (shipping/discount), most-specific Google Product Category, primary brand color (HEX, for minimal compliant background), accent color (HEX, sale-badge only, small/non-intrusive).

## Section 2 — technical specs (compliance-driven)
Primary image ratio 1:1 (1200×1200 min) — required for Shopping listing format. Secondary ratio 1.91:1 (1200×628 min) — Display/Demand-Gen/PMax horizontal asset. Primary (first) product image background must be **pure white (#FFFFFF)** — mandatory for Shopping listing compliance. Text/logo overlay **strictly forbidden on the primary image**; allowed only on secondary/lifestyle assets, max 20% coverage. File ≤5MB, JPEG/PNG. Apparel needs a fully-visible centered model on concepts A/B/E.

## Section 3 — 5 creative concepts (psychological angle × primary/secondary split)
Each concept = 1 compliant white-bg primary Shopping image (no text) + 1 secondary 1.91:1 lifestyle/overlay asset for PMax/Display:

- **A — Aspirational Lifestyle:** psychology = aspiration (identify with the outcome). Primary: clean 3/4-view white-bg product shot. Secondary: model using product in a desirable minimal setting (e.g. high-end cafe / scenic vista), shallow depth of field, product in focus.
- **B — Product-Feature Deep Dive:** psychology = perceived value/rationality (justifies price via engineering). Primary: white-bg, cropped to the unique feature (sole/material/weave). Secondary: extreme macro of the feature in action (e.g. water beading off a waterproof surface), harsh studio light, technical-drawing feel.
- **C — Social Proof / Validation:** psychology = bandwagon effect. Primary: standard full white-bg shot. Secondary: neutral dark-gray background, high-contrast accent-color badge stating rating + review count ("4.8/5.0 — Trusted by 10,000+ Customers"), product stays focal point.
- **D — Price Anchoring:** psychology = loss aversion (temporary win framing). Primary: standard white-bg shot. Secondary: sale badge overlay (accent color) with small crossed-out MSRP + large bold current price, urgent-but-professional mood.
- **E — Problem/Solution Contrast:** psychology = pain avoidance. Primary: standard white-bg shot. Secondary: vertical split-screen — dark/blurred "before" (old inferior solution) vs bright "after" (product solving it cleanly), sharp dividing line.

Each concept card also specifies: primary offer reinforcement line (e.g. free shipping / lifetime warranty / satisfaction guarantee / limited-time / risk-free trial) and a targeted CTA button (Shop Now / Learn More / Read Reviews / Claim Offer / Discover Comfort) matched to that concept's psychological angle.
