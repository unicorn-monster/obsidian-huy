## Setup
Native image ads run in Demand Gen: YouTube in-feed, YouTube search/home/watch-next, and Google Discover. YouTube in-feed = thumbnail advertising — ad competes with organic YouTube thumbnails, need contrast/curiosity to make someone pause. Discover = content-card environment — ad needs to feel like it belongs in the feed (blog/article style).

Principle: image must fit the problem → headline must fit the image → landing page must fit the headline. Build in **clusters**: one cluster per angle (informational / comparison / offer), each with its own image styles, headlines, descriptions, and landing page. Don't mix intents into one ad set — mixed signals = messy data, harder for Google to optimize.

How to pick an angle for your category: (1) Google Keyword Planner — high volume on informational queries ("best magnesium for sleep", "relief from muscle cramps") signals informational/comparison angles will convert; (2) existing Search campaigns' Search Terms Report — filter for "how to", "vs", "best" to see what frame of mind prospects are in.

## The 3 angles

**1. Informational** — built around a problem/outcome/myth/mistake/education. Goal is curiosity + click, not immediate purchase; headline opens a loop, never says "buy this."
- Headline example: "Doctors Explain The Hidden Reason Why Most Knee Pain Treatments Don't Work"
- Image directions: (a) problem-focused visual — someone holding their knee, wincing on stairs; (b) clinical/editorial — anatomy close-up, doctor-style image, works especially well on Discover (feels like article content); (c) curiosity-driven — unusual visual that doesn't give the answer away, image should support curiosity, not solve it.
- Description examples: "Discover the approach helping people recover faster and feel better day to day." / "Based on methods trusted by professionals and experts." / "For people who want to stay active, mobile, and pain-free for longer."
- Landing page: **advertorial or quiz** (never straight to product page — user clicked for information, so the LP must continue the education before pitching product as the solution).
- Funnel: informational ad → advertorial/quiz → product.

**2. Comparison** — built around rankings, reviews, "best of" lists, product comparisons. Works because people want a shortcut / want to know the best option before spending money; Google's in-market audiences target these people well.
- Headline example: "Top Chefs in the World Review 5 Pans" (authority + specific number + curiosity about what made the list).
- Image directions: editorial product layout with multiple products together; lifestyle images showing evaluation happening; side-by-side comparisons with clear visual hierarchy.
- Landing page: **comparison/ranked review page** (your product ranks #1) → click into product → product page.
- Funnel: comparison ad → ranked review/comparison page → product page.

**3. Offer-focused** — most direct angle, built purely around the deal. Works best when the audience already understands the product/category/brand (warmer audiences) — essentially what most brands already run on Meta.
- Headlines: "Save 25% Today", "Limited Time Bundle Offer", "Free Shipping This Week."
- Description reinforces urgency: "Limited inventory available", "Offer ends soon."
- Landing page: **dedicated sale/bundle page** that shows the deal immediately — any mismatch between ad promise and LP creates friction.
- Funnel: offer ad → sale/bundle page directly.

## Placement nuance
Same angle performs differently by placement — build 2 versions of each angle and let data decide, don't rely on instinct:
- YouTube in-feed: stronger problem visuals with more contrast (thumbnail-scanning behavior).
- Google Discover: editorial/article-style images work better (feels like content, not ad).

## Campaign structure & scaling
- One campaign per angle × primary placement — never mix angles/placements into one campaign. Name clearly so the account stays readable; each campaign has one job, so you can diagnose (e.g. informational getting clicks but not converting → the advertorial needs work, not the ad).
- Demand Gen assets: up to 20 images, 5 headlines, 5 descriptions per ad — Google auto-combines them, so all assets within a cluster must stay in the same intent lane (don't mix a product photo + offer headline + vague description + send everything to the same product page).
- Targeting: start broad, feed Google interest/demographic signals, don't over-restrict.
- Scaling rule: when something wins, increase budget ~20% every 3-5 days as long as performance holds. Demand Gen creatives are volatile — treat every win as the start of a new creative test, not a "set and forget."
