## 20 image prompts (4 hook × 5 ảnh RIÊNG) — DETAILED, target 8/10+

> [!important] Vì sao 2 ảnh anh gen ra SAI (windowsill / cục pin): prompt NGẮN + chữ "grinder" trơ → AI vẽ **grinder cỏ/cà phê** hoặc cục pin vô nghĩa. Fix = prompt DETAILED + KHOÁ subject + NEGATIVE.
> **Anatomy 1 prompt:** `[amateur framing] + [SUBJECT chính xác + NEGATIVE] + [state/problem] + [scene nhà thật] + [lighting xấu] + [imperfection] + no text + Square 1:1.`
> **Khoá subject (bắt buộc):**
> - Máy mài móng = *"a small handheld electric **DOG NAIL grinder** — white/grey plastic pen-shaped cordless rotary tool the size of an electric toothbrush, spinning sanding-drum tip + clear nail-guard cap. **NOT a herb/weed/coffee/angle grinder, not a flashlight.**"*
> - Móng chó = *"a real dog's paw, [size], **thick black / overgrown** nails [curling into the pad / splaying the toes]."*
> **Luật 8+:** lead bằng **pain / dread / guilt**. Shot đồ-vật-đơn (grinder hỏng) trần ~7 dù detailed. Avg set ~8.0.
